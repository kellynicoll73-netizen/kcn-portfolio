# Carousel mouseover copy — kellynicoll.ca

These six blocks of copy are for the image carousel on the homepage of kellynicoll.ca. Each slide in the carousel corresponds to one case study or the thought leadership piece.

When a user hovers over a carousel slide, a text overlay should appear on top of the image containing three elements in this order:

1. **Category** — small uppercase label, tertiary/muted treatment
2. **Title** — the name of the case study or piece, bold
3. **Director-level highlight** — italic, one line, establishes Director-level context for a reader who has not yet read the piece
4. **Teaser** — one or two sentences summarising the key Director-level credential and outcome

The overlay should be subtle — semi-transparent dark background, white text, appearing on hover with a smooth fade transition. It should not obscure the image entirely.

## Notes for Code

- Slides should auto-advance on a timer (suggested: 5 seconds per slide)
- Manual prev/next navigation via arrow buttons inset within the left and right edges of the carousel image
- Dot navigation at the bottom of the carousel showing current position in the stack
- Each slide image should link through to the relevant case study page or the Thinking page on click
- The hover overlay should fade in smoothly — avoid hard cuts
- On mobile, the overlay text should be always visible (no hover state on touch devices)
